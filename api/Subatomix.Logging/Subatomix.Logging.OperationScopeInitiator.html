    Class OperationScopeInitiator

    Begins or performs operations in an OperationScope with a specific Microsoft.Extensions.Logging.ILogger, Microsoft.Extensions.Logging.LogLevel, and operation name.

    Inheritance
    System.Object
    OperationScopeInitiator
    Implements
    IFluent
    Namespace: Subatomix.Logging
    Assembly: Subatomix.Logging.dll
    Syntax
    public sealed class OperationScopeInitiator : ValueType, IFluent
    Remarks

    This type is part of a fluent API. Obtain instances of this type via Operation(ILogger, String) or Operation(ILogger, LogLevel, String).
